<Text>Правила определяют механизм подмены номеров на сайте: количество номеров коллтрекинга, условия их показа и переадресация звонков на эти номера.</Text>
<Space 8/>
<Text>В названии правила удобно указать сайт, площадку или сегмент посетителей, на который оно настроено.</Text>
<Space 16/>
<SubTitle>Динамический коллтрекинг</SubTitle>
<Space 8/>
<Text>Связывает звонки с сессиями конкретных пользователей, получая максимум информации о звонящих.</Text>
<Space 16/>
<Image src="https://github.com/Scadar/onboarding_md/blob/main/static/call-tracking-pkt-img1.png?raw=true" alt="" height="196px"/>
<Space 16/>
<Text>Каждый посетитель видит номер телефона, который в этот момент демонстрируется только ему.</Text>
<Space 16/>
<Container><Text>Рекомендуем настроить динамический коллтрекинг, который будет отслеживать всех посетителей</Text></Container>
<Space 16/>
<Text>Если необходимо отслеживать только определенные каналы и источники, выберите их из списка. Также вы можете создать свой собственный рекламный источник или использовать ручную настройку для тонкой сегментации трафика.</Text>
<Space 16/>
<SubTitle>Статический коллтрекинг</SubTitle>
<Space 8/>
<Text>Отслеживает факт звонка из выбранных источников, не связывая его с конкретными сессиями.</Text>
<Space 16/>
<Image src="https://github.com/Scadar/onboarding_md/blob/main/static/call-tracking-pkt-img2.png?raw=true" alt="" height="224px"/>
<Space 16/>
<Text>Выберите «Статический коллтрекинг» → «Подмена номера на сайте» для отслеживания звонков с сайта. Номера будут подменяться для посетителей из источников, которые будут выбраны на следующем шаге.</Text>
<Space 16/>
<Container><Text>Посетители из источника А увидят номер телефона 1. Все звонки на номер 1 будут связаны с источником А.</Text></Container>
<Space 16/>
<Text>Выберите «Звонки со сторонних площадок» для отслеживания звонков с групп в соцсетях, маркетплейсов, уличных баннеров и т.п.</Text>
<Space 16/>
<Container><Text>На площадке Б размещен номер 2. Все звонки на номер 2 будут связаны с этой площадкой.</Text></Container>
